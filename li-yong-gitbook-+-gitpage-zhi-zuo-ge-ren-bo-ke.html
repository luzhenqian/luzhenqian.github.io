利用gitbook+gitpage制作个人博客

首先安装npm环境。

npm i -g gitbook

安装完后可以检测是否正常安装：gitbook -v

下载gitbook editor

地址：https://legacy.gitbook.com/editor

使用gitbook editor写完的书可以复制到其他地方。在书的根目录下使用gitbook init命令初始化gitbook，之后使用gitbook serve启动服务，也可以使用gitbook build生成静态网页。

gitbook安装插件。

在根目录下创建book.json文件，写入以下代码。

{
  "plugins": [
    "back-to-top-button"
  ]
}

执行gitbook install

在book.json文件中的所有插件都会被安装上。

gitbook没办法卸载插件，只能修改book.json文件，再执行一次gitbook install

插件地址：https://plugins.gitbook.com/
